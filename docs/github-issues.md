# GitHub Issues 起票案

対象: AI 面接練習プロトタイプ

現状は静的 HTML/CSS/JavaScript と `localStorage` で動くブラウザ内プロトタイプです。実 AI 連携、Google/Firebase、クラウド同期、ファイル取込、本番向けバックエンドは未実装です。

## 現在の実装済み範囲

- 画面: アカウント、企業・ES管理、面接設定、面接チャット、最終フィードバック、履歴。
- 保存: `localStorage` にアカウント、企業、ES、面接ログ、選択中アカウントを保存。
- ES管理: 企業登録、ES設問・回答・文字数・カテゴリ・ステータス保存、文字数超過チェック。
- ES起点の面接: 保存済み ES を選んで面接設定へ反映し、質問生成に ES 情報を渡す。
- 面接進行: 質問数設定、回答送信、深掘り質問、会話タイムライン、途中終了。
- 評価: 文字数・構造語・具体語・企業名/職種名などを使ったモック採点。
- 最終FB: 総合点、評価軸別スコア、良い点、改善点、次回練習項目。
- 履歴: 面接ログの保存、一覧、詳細表示、個別削除、全削除。
- CSS: レスポンシブ対応済み。

## 推奨マイルストーン

1. プロトタイプ安定化
2. 実 AI 連携
3. アカウント・クラウド保存
4. ES 分析・ファイル取込
5. 本番化・運用準備
